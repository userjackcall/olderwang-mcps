# olderwang-mcps

老王自製的 OpenCode MCP 伺服器集合。

## 目錄

| MCP | 說明 |
|-----|------|
| [agnes-image](agnes-image/README.md) | AI 圖片生成 — 呼叫 Agnes AI 模型，支援文生圖與圖生圖 |
| [gas-mcp](gas-mcp/README.md) | Google Apps Script 全功能管理 — 檔案讀寫、程式執行、部署、Git 版控、試算表查詢 |

## 用法

各 MCP 獨立存放於其子目錄中，每個目錄內有完整的 README 說明安裝、設定與使用方式。
